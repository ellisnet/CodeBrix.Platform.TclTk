NOTE ON THE Tcl LANGUAGE

CodeBrix.Platform.TclTk implements the Tcl scripting language. "Tcl" refers to the
Tool Command Language originally created by John Ousterhout. CodeBrix.Platform.TclTk
is a managed, from-source implementation (via the Eagle project above) and does NOT
incorporate the source code of the reference native Tcl/Tk distribution.
